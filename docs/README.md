## Change Color Theme

To alter the color scheme of a component, update the palette properties on the <em>parent component</em> so that all children are affected.

**button**:
+ `palette.button`
+ `palette.buttonText`

**window & base**:
+ `palette.base`
+ `palette.text`
+ `palette.window`
+ `palette.windowText`

**highlights**:
+ `palette.highlight`
+ `palette.highlightedText`
